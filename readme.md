# eJPT Notes

A personal collection of notes, commands, and concepts gathered while preparing for the eLearnSecurity Junior Penetration Tester (eJPT) certification.

These notes are written in **English, Spanish and French**.

## Contents
- Networking basics
- Enumeration techniques
- Exploitation fundamentals
- Web application testing
- Useful tools and commands

## Purpose
This repo serves as a quick reference and study guide, focused on practical understanding rather than theory-heavy explanations.

## Status
This repository is **under construction** and will evolve over time. It may eventually be turned into a GitBook or updated continuously as the material grows and improves.

## Disclaimer
These notes are for educational purposes only. Use responsibly and only on systems you have permission to test.
